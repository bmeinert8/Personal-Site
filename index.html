Personal Site

In this project I created a personal Website to host my resume, certifications, and information about myself. Once the website was created I uploaded the documents to Azure to host the site in a static web storage container. Once it was uploaded on Azure, I had already purchased a domain name, so I created an Azure CDN profile and DNS service to host my domain through Azure. You can find my blog post on the creation process of the CDN and DNS here. Finally, I integrarted CI/CD with github actions to contiously keep this site up to date, as I build more projects and earn more certifications.
